# justify-content

**Since** UXP v3.0

Supported values are `flex-start`, `flex-end`, `center`, `space-between`, `space-around`, `stretch`.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/justify-content

**Example**

```css
.someElement {
    justify-content: flex-start;
}
```
