# :nth-child

**Since:** UXP 3, PS 22.0.0

Matches only the Nth child. Can use an expression or `even` or `odd`.

**See:** https://developer.mozilla.org/en-US/docs/Web/CSS/:nth-child

**Example:**

```css
.row:nth-child(even) {
    background-color: #E8E8E8; /* color even rows */
}
```
