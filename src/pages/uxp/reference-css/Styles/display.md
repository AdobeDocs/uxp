# display

**Since:**  UXP 3

Determines the layout semantics for the element. Supports `none`, `inline`, `block`, `inline-block`, `flex`, and `inline-flex`.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/display

**Example**:

```css
.someElement {
    display: flex;
}
```
