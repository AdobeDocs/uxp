# margin-right

**Since:**  UXP 2, PS 22.0.0

Specifies the right margin for an element.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/margin-right

**Example**:

```css
.someElement {
    margin-right: 1em;
}
```

## Quirks and Exceptions

* Margins do not collapse in UXP.
