# white-space

**Since:**  UXP 3, PS 22.0.0

Determines how white space is handled. Supports `wrap` (default), `nowrap`, `pre`, `pre-line`, and `pre-wrap`.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/white-space

**Example**:

```css
.someElement {
    white-space: pre;
}
```
