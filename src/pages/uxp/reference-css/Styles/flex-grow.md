# flex-grow

**Since:** UXP 3, PS 22.0.0

Determines if a flexible item can grow, and if so, how it grows relative to other items in the flexible container.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/flex-grow

**Example**

```css
.someElement {
    flex-grow: 2;
}
```
