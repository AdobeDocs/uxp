# margin-left

**Since:**  UXP 2, PS 22.0.0

Specifies the left margin for an element.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/margin-left

**Example**:

```css
.someElement {
    margin-left: 1em;
}
```

## Quirks and Exceptions

* Margins do not collapse in UXP.
