# html

**Since:** UXP 2, PS 22.0.0

Indicates the root element of an UXP plugin's UI.

**See**: https://developer.mozilla.org/en/docs/Web/HTML/Element/html

**Example**

```html
<html>
    <head>
    </head>
    <body>
        Hello, world
    </body>
</html>
```

## Attributes

The following attributes are not supported, and have no effect:

* `manifest` 
* `version`
* `xmlns`
