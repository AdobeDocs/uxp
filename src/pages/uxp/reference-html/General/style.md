# style

**Since:** UXP 2, PS 22.0.0

Specifies an inline stylesheet.

**See**: https://developer.mozilla.org/en/docs/Web/HTML/Element/style

**Example**

```html
<style>
    .highlight { color: red; }
</style>
```

## Attributes

The following attributes are unsupported and ignored:

* `type`
* `media`
* `nonce`
* `title`
